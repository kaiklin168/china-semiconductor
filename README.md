# 🇨🇳 中国半导体专属 Repo

中国半导体产业新闻收集与股票分析自动化系统。

## 📋 功能概览

| 模块 | 功能 | 状态 |
|------|------|------|
| 新闻收集 | 从多个来源收集中国半导体相关新闻 | ✅ 运行中 |
| 股票分析 | A股/港股半导体股票深度分析 | ✅ 运行中 |
| Telegram 推送 | 自动推送分析报告到 Telegram | ✅ 运行中 |
| Git 同步 | 自动同步数据到 GitHub | ✅ 运行中 |

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install requests beautifulsoup4 playwright pandas numpy yfinance
python -m playwright install chromium
```

### 2. 配置 Telegram Bot

在脚本中配置 Bot Token 和 Chat ID（参考你的配置）

### 3. 运行新闻收集

```bash
python news/news_collector_cn.py "半导体设备" "国产替代" "中芯国际"
```

### 4. 运行股票分析

```bash
python analysis/cn_stock_analysis.py 688256
```
